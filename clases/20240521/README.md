### Super Geostrofico.

Fuerzas que se equilibran.
- Gradiente de presion, no lo puedo cambiar, es constante.
- Coriolis
- Viento que se mueve
- Fuerza centrifuga, va para afuera. es constante no lo puedo cambiar.

En una alta, se incrementa agranda el viento.
